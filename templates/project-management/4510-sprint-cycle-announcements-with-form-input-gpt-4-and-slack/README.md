# 📋 Sprint cycle announcements with Form Input, GPT-4 and Slack

> ⚡ **273 views** · 📋 [Project Management](../)

## Description

How it works
- Triggers on submitting an n8n form
- Uses the form details to prepare a message
- Sends the message to Slack

Set up Steps
- Add in your team name
- Add in message tone
- Set up Open AI
- Set up Slack

## 🔗 Nodes Used

Slack, AI Agent, OpenAI Chat Model, n8n Form Trigger

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
