Whether you're automating your first workflow or building complex multi-step integrations, this collection has you covered. Every template is a `.json` file you can import directly into [n8n](https://n8n.io) in seconds.

---

## ⚡ Quick Start

Getting started takes less than a minute:

1. **🔍 Browse** the [categories](#-categories) below to find what you need
2. **📥 Download** the `.json` template file
3. **🚀 Import** into n8n: `Workflow menu → Import from File`

That's it — your automation is ready to configure and run! 🎉

> 📚 **New to n8n?** Check out our [step-by-step importing guide](docs/importing-templates.md) and [credential setup walkthrough](docs/credential-setup.md).
