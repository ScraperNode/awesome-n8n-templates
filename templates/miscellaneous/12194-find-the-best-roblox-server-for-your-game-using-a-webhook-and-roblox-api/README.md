# 🔧 Find the best Roblox server for your game using a webhook and Roblox API

> ⚡ **79 views** · 🔧 [Miscellaneous](../)

## Description

# Who For?
Gamers who don't like bad and slow preformance playing games and want to find a good preformance based server near them.

# Setup?
None!

# Modes Available 
- Auto - Optimized (Recommended & Default)
- Ping - Finds the lowest ping
- Latency - Lowest ping & highest FPS

## 🔗 Nodes Used

HTTP Request, Webhook
