# 📱 Send bulk messages to chats in Telegram

> ⚡ **10,053 views** · 📱 [Social Media & Email Marketing](../)

## Description

The Telegram API has a [limitation](https://core.telegram.org/bots/faq#broadcasting-to-users) to send only 30 messages per second. 

Use this workflow to send messages to more than 30 chats in Telegram.

## 🔗 Nodes Used

Google Sheets, Start, Telegram

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
