# 📱 Bulk emails with personalized attachment

> ⚡ **5,056 views** · 📱 [Social Media & Email Marketing](../)

## Description

A workflow to send personalized emails with respective attachment.


The workflow needs a pre-formatted CSV with file names and email IDs.

## 🔗 Nodes Used

Send Email, Read Binary File, Spreadsheet File, Start

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
