# ⚒️ Get execute command data and transfer to JSON

> ⚡ **2,801 views** · ⚒️ [Engineering](../)

## Description

Shows how it is possible to use the data of a command line tool which returns JSON.

Example shows how:
1. to bring data in flow
2. to use data directly in a node

Note that the 'execute command' node is not available on n8n Cloud.
