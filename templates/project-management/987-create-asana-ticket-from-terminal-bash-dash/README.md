# 📋 Create Asana ticket from terminal bash-dash

> ⚡ **889 views** · 📋 [Project Management](../)

## Description

This workflow allows creating a new Asana task via [bash-dash](https://github.com/n8n-io/bash-dash) 

![workflow-screenshot](fileId:445)

**Example usage:**
\- asana My new task

**Example bash-dash config:**
commands[asana]="http://localhost:5678/webhook/asana"

## 🔗 Nodes Used

Asana, Webhook

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
