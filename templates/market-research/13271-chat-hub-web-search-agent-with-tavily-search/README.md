# 📊 Chat Hub Web Search Agent with Tavily Search

> ⚡ **1,346 views** · 📊 [Market Research & Insights](../)

## Description

A blazing fast research agent that works nativly in n8n's [Chat Hub](https://www.youtube.com/watch?v=s_GHteEnHc4).

This agent is preconfigured with an optional system prompt for both Chat Hub and to use the Tavily search tool.

### Setup Steps
1. Sign up for the generous Tavily [Free Tier](https://www.tavily.com/pricing)
2. Add your Tavily API key in the credentials in the tool
3. Never use a search engine again!

## 🔗 Nodes Used

AI Agent, Simple Memory, Chat Trigger, OpenRouter Chat Model

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
