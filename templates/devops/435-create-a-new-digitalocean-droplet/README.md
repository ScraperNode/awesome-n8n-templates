# ⚙️ Create a new DigitalOcean droplet

> ⚡ **1,296 views** · ⚙️ [DevOps & CI/CD](../)

## Description

Uses a DigitalOcean Personal Access Token to create a new Droplet.

Add your personal access token and change the parameters to create droplets of different sizes.

You can also specify more options; refer the [API docs](https://developers.digitalocean.com/documentation/v2/#create-a-new-droplet)

## 🔗 Nodes Used

HTTP Request

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
