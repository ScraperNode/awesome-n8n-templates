# ⚙️ Push your public IP to Namecheaps dynamic DNS

> ⚡ **1,089 views** · ⚙️ [DevOps & CI/CD](../)

## Description

Template to get your public IP address and push it to Namecheaps Dynamic DNS per subdomain.

1. Open "yourdomain.com"
2. Insert your domain and your Namecheap DDNS password
3. Open "subdomains"
4. Replaces and insert your subdomains
5. Execute Workflow

Have fun!

## 🔗 Nodes Used

Cron, Function, HTTP Request, Start

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
