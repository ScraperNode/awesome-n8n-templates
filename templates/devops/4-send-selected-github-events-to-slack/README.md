# ⚙️ Send selected GitHub events to Slack

> ⚡ **3,651 views** · ⚙️ [DevOps & CI/CD](../)

## Description

![workflow-screenshot](fileId:5)

1. Gets triggered by GitHub once a user stars/unstars a repository
1. Checks if star got added or removed and forwards data depending on that
1. Sends notification to Slack

## 🔗 Nodes Used

Github Trigger, Slack

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
