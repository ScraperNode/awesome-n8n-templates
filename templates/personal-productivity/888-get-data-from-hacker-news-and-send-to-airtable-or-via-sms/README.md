# ⚡ Get data from Hacker News and send to Airtable or via SMS

> ⚡ **2,652 views** · ⚡ [Personal Productivity](../)

## Description

This n8n workflow automates sending out SMS notifications via Vonage which includes new tech-related vocabulary everyday.


To build this handy vocabulary improver, you’ll need the following:

-   n8n – You can find details on how to install n8n on the [Quickstart](https://docs.n8n.io/getting-started/quickstart.html) page.
-   LingvaNex account – You can create a free [account](https://lingvanex.com/registration/) here. Up to 200,000 characters are included in the free plan when you generate your API key.
-   Airtable account – You can [register](https://airtable.com/signup) for free.
-   Vonage account – You can [sign up](https://dashboard.nexmo.com/sign-up) free of charge if you aren’t already.
