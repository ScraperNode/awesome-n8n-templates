## Description

Automated pipeline that exports technology stack data from BuiltWith to Google Sheets for analysis, reporting, and team collaboration.

## 🚀 What It Does

- Extracts technology stack data
- Organizes data in Google Sheets
- Updates automatically on schedule
- Supports multiple company tracking
- Enables easy data sharing
