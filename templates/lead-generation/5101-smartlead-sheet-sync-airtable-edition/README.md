## Description

Auto-capture client inquiries straight into Airtable —  no coding, just plug & play.

Adds each new lead into your Airtable base — like magic

Freelancers & coaches

Creators with landing pages or service funnels

A pre-built n8n workflow (Webhook → Code → Airtable)

Clean, editable JavaScript code to match your form fields
Thumbnail for marketing or reselling
