# 🎣 Perform an email search with Icypeas (single)

> ⚡ **971 views** · 🎣 [Lead Generation & Enrichment](../)

## Description

This template can be used to search for an email address with [Icypeas](https://www.icypeas.com/). Be sure to have an active account to use this template. 



**How it works**
This workflow can be divided into three steps :
1. The workflow initiates with a manual trigger (On clicking 'execute').
2. It connects to your Icypeas account.
3. It performs an HTTP request to search for an email address.

**Set up steps**
1. You will need a working icypeas account to run the workflow and get your API Key, API Secret and User ID.
2. You will need a personn firstname, lastname and domain/company name to perform the search.

## 🔗 Nodes Used

HTTP Request
