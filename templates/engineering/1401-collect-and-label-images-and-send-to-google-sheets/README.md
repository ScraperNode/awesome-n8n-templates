# ⚒️ Collect and label images and send to Google Sheets

> ⚡ **1,449 views** · ⚒️ [Engineering](../)

## Description

This workflow collects images from web search on a specific query, detects labels in them, and stores this information in a Google Sheet.

## 🔗 Nodes Used

Google Sheets, HTTP Request, AWS Rekognition

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
