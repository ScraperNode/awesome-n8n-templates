# 📁 Convert image URLs to an uploaded attachment in Airtable

> ⚡ **2,360 views** · 📁 [File Management](../)

## Description

Super simple workflow to convert image URLs to an uploaded attachment in Airtable.

You'll need to adjust the field names to match your specific data, including in the filter formula where it says "Cover image URL". Just replace that with the field name where you are storing the image URL.

## 🔗 Nodes Used

Airtable

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
