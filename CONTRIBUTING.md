# Contributing to Awesome n8n Templates

Thanks for your interest in contributing! Here's how you can help grow this collection.

## Adding a Template

1. Fork this repository
2. Add your workflow JSON file to the appropriate category folder under `templates/`
3. Use kebab-case for file names (e.g., `my-workflow-name.json`)
4. Open a pull request with a brief description of what the template does

## Template Guidelines

- Templates should be **working workflows** exported from n8n
- Remove any personal credentials or API keys before submitting
- Use descriptive names that explain what the workflow does
- Include a description in the workflow metadata if possible

## Suggesting a Category

If your template doesn't fit any existing category, open an issue to suggest a new one.

## Reporting Issues

- If a template doesn't work, open an issue with the template name and the error you encountered
- If you find duplicate templates, let us know

## Code of Conduct

Be respectful and constructive. We're all here to make n8n automation more accessible.
