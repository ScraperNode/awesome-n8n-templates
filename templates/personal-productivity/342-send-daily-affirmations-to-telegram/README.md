# ⚡ Send daily affirmations to Telegram

> ⚡ **2,524 views** · ⚡ [Personal Productivity](../)

## Description

![telegramworkflow.PNG](fileId:30)

A workflow which allows you to receive daily affirmations via Telegram by querying a REST API triggered by a Cron node. 


I used the [affirmations.dev](https://github.com/annthurium/affirmations) API

## 🔗 Nodes Used

Cron, HTTP Request, Telegram

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
