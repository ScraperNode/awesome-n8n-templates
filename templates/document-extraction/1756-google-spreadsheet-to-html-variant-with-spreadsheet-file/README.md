# 🔬 Google spreadsheet to HTML - variant with spreadsheet file

> ⚡ **2,755 views** · 🔬 [Document Extraction & Analysis](../)

## Description

This workflow shows a no code approach to creating a HTML table based on Google Sheets data.

![image.png](fileId:653)

To run the workflow:
1. Make sure you have a Google Sheet with a header row and some data  in it.
2. Grab your sheet ID:
![image.png](fileId:654)
3. Add it to the Google Sheets node:
![image.png](fileId:655)
4. Activate the workflow or execute it manually
5. Visit the URL provided by the webhook node in your browser (*production URL* if the workflow is active, *test URL* if the workflow is executed manually)

## 🔗 Nodes Used

Google Sheets, Spreadsheet File, Webhook

Download [`workflow.json`](workflow.json) and import into n8n:
