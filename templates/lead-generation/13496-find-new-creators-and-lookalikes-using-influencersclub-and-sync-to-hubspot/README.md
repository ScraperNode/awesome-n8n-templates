## Description

How it works:

Discover and filter creators with social platform data and find lookalikes to add to your CRM for influencer marketing platforms or creator outreach.

Step by step workflow to discover new creators with multi social (Instagram, Tiktok, Youtube, Twitter, Onlyfans, Twitch and more) profiles, analytics and metrics using the influencers.club API, find similar ones and create/update records in Hubspot

Set up:

Hubspot (can be swapped for any CRM like Salesforce or Google Sheet)
Influencers.club API

## 🔗 Nodes Used

HubSpot, Schedule Trigger, AI Agent, OpenAI Chat Model, Structured Output Parser
