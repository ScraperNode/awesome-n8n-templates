# 🔬 Receive Google Sheet data via REST API

> ⚡ **7,769 views** · 🔬 [Document Extraction & Analysis](../)

## Description

![workflow-screenshot](fileId:28)

Simple workflow which allows to receive data from a Google Sheet via "REST" endpoint.

1. Wait for Webhook Call
1. Get data from Google Sheet
1. Return data


Example Sheet: [https://docs.google.com/spreadsheets/d/17fzSFl1BZ1njldTfp5lvh8HtS0-pNXH66b7qGZIiGRU](https://docs.google.com/spreadsheets/d/17fzSFl1BZ1njldTfp5lvh8HtS0-pNXH66b7qGZIiGRU)

## 🔗 Nodes Used

Google Sheets, Webhook
