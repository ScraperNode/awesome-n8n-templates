# 🎬 Generate and publish Instagram Reels with Gemini and Veo

> ⚡ **311 views** · 🎬 [Content Creation & Video](../)

## Description

### How it Works

* The workflow runs automatically on a schedule (daily at a set time).
* It defines the video topic, caption, and aspect ratio.
* Gemini generates a detailed visual prompt for a short-form vertical video.
* Google Veo creates the video based on that prompt.
* The finished video is published to Instagram as a Reel with the preset caption.

---

### Set-up Steps

* Connect Google Gemini and Veo API credentials.
* Authenticate Instagram publishing access.
* Customize the video topic, caption, and posting schedule.
* Run a quick test to verify everything works.

**Estimated setup time:** ~20–40 minutes.

## 🔗 Nodes Used

Schedule Trigger, Google Gemini
