## Description

## 🏥 PATIENT APPOINTMENT REMINDER SYSTEM
## MULTI-DATABASE SUPPORT

**PURPOSE:**
Automatically send professional appointment reminders via email and SMS to reduce no-shows and improve patient experience.

**FEATURES:**
✅ Dual reminders (3-day + 1-day before)
✅ Email + SMS notifications
✅ Phone number auto-formatting
✅ Timezone handling
✅ Professional HTML email templates
✅ Multi-database support:
   • PostgreSQL (Enterprise)
   • Google Sheets (Easy)
   • Airtable (Recommended)
✅ Flexible webhook support

**BUSINESS IMPACT:**
📈 Reduce no-shows by 30-50%
💰 Increase revenue from better attendance
⏰ Save staff time on manual reminders
📱 Professional patient communication

**SETUP:**
Activate ONLY the database nodes your client needs!

## 🔗 Nodes Used

Airtable, Send Email, Google Sheets, Postgres, Twilio, Webhook
