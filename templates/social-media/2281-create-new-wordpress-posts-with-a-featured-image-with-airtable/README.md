# 📱 Create new WordPress posts with a featured image with Airtable

> ⚡ **12,949 views** · 📱 [Social Media & Email Marketing](../)

## Description

This workflow is aimed to create new posts in wordpress automatically from an airtable dashboard.
When creating content in bulk, we can save time by automating how we can post and publish this content.

## Usage
1. Get the content from Airtable. Since we have this as a markdown, we will have to convert it to a html format to make it easier to publish and manage on WordPress
2. Upload the blog post with the content, title and all other relevant information needed for an optimized blog
3. Once the post is posted, we need to upload the image and set it as a features image for the blogs

Happy productivity

## 🔗 Nodes Used

Airtable, HTTP Request, Wordpress, Markdown, Schedule Trigger, Filter

Download [`workflow.json`](workflow.json) and import into n8n:
