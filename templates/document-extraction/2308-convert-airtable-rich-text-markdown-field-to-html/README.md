# 🔬 Convert Airtable rich text markdown field to HTML

> ⚡ **656 views** · 🔬 [Document Extraction & Analysis](../)

## Description

Sometimes you need the rich text field to be in HTML instead of Markdown. This template either syncs a single record or all records at once.

[Youtube tutorial](https://www.youtube.com/watch?v=PAoxZjICd7o)

## 🔗 Nodes Used

Airtable, Webhook, Markdown

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
