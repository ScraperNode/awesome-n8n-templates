# 🤝 Validate website of new companies in HubSpot

> ⚡ **1,505 views** · 🤝 [CRM & Sales Operations](../)

## Description

This workflow uses a Hubspot Trigger to check for new companies. It then checks the companies website exists using the HTTP node. If it doesn't, a message is sent to Slack.

To configure this workflow you will need to set the credentials for the Hubspot and Slack Nodes. You will also need to select the Slack channel to use for sending the message.

## 🔗 Nodes Used

HTTP Request, Slack, Start, HubSpot, HubSpot Trigger
