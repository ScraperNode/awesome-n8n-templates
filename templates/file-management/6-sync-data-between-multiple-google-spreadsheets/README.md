# 📁 Sync data between multiple Google Spreadsheets

> ⚡ **4,468 views** · 📁 [File Management](../)

## Description

![workflow-screenshot](fileId:8)

1. Triggers workflow all two minutes
1. Reads data from a Google Spreadsheet (in example Sheet Data columns A to G)
1. Write the data unchanged in two different Spreadsheets with same Sheet name and columns, expressions are optional)

## 🔗 Nodes Used

Cron, Google Sheets

## 📥 Import

Download [`workflow.json`](workflow.json) and import into n8n:
**Workflow menu → Import from File**
