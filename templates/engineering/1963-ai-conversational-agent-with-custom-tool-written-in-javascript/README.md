# ⚒️ AI: conversational agent with custom tool written in JavaScript

> ⚡ **54,000 views** · ⚒️ [Engineering](../)

## Description

This workflow implements a custom tool via JavaScript code which returns a random color to users and excludes the given colors.

Note that to use this template, you need to be on n8n version 1.19.4 or later.

## 🔗 Nodes Used

AI Agent, OpenAI Chat Model, Code Tool, Chat Trigger
