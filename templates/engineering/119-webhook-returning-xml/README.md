# ⚒️ Webhook returning XML

> ⚡ **3,779 views** · ⚒️ [Engineering](../)

## Description

![workflow-screenshot](fileId:20)

1. Receives data from an incoming HTTP Request (set up to use respond to webhook node)
1. Create dummy data
1. Convert JSON to XML which gets returned 
1. Respond to Webhook which returns the data and the content type of the data
