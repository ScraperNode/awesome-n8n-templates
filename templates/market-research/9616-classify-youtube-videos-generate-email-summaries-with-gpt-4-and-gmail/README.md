## ЁЯзн How It Works (Node Map)
1. **Manual Run** тАУ adтАСhoc execution  
2. **Set Channel IDs** тАУ provide one or more YouTube `channelId` values  
3. **Split Channels** тАУ process channels one by one  
4. **Fetch Latest Videos (RSS)** тАУ pull recent uploads via channel RSS  
5. **Filter: Published in Last 72h** тАУ only recent items are kept  
6. **Get Video Stats (YouTube API)** тАУ request `snippet,statistics` for likes and details  
7. **Classify by Likes (Code)** тАУ sets `classification` to `viral` or `normal`  
8. **Branch: Normal / Branch: Viral** тАУ separate LLM prompts per relevance  
9. **Write Post (Normal / Viral)** тАУ generate LinkedInтАСstyle notes via OpenAI  
10. **Aggregate Posts for Briefing** тАУ merge all texts into one block  
11. **Generate Weekly Briefing (HTML)** тАУ produce a GmailтАСrobust HTML email via GPT  
12. **Send Weekly Briefing (Gmail/SMTP)** тАУ deliver briefing (you set recipients)

## ЁЯзк Troubleshooting
- **Missing `likeCount` / `classification = "unknown"`** тЖТ ensure `part=statistics` and a valid API key credential.  
- **Gmail OAuth `redirect_mismatch` / `access_denied`** тЖТ redirect must be `https://&lt;your-n8n-host&gt;/rest/oauth2-credential/callback` and test users added if restricted.  
- **SMTP auth issues** тЖТ set correct server/port/TLS and use an app password when 2FA is enabled.  
- **Empty LLM output** тЖТ verify OpenAI key/quota and inspect node logs.
